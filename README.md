## 🧩 Featured Project

### 👶 Babysitter Recommendation System (FYP)

<p align="center">
  <img src="https://img.shields.io/badge/Architecture-Microservices-blue?style=flat-square"/>
  <img src="https://img.shields.io/badge/Realtime-WebRTC-green?style=flat-square"/>
  <img src="https://img.shields.io/badge/Backend-NestJS-red?style=flat-square"/>
</p>

A full-stack platform connecting mothers and babysitters using intelligent matching and real-time communication.

**Core Features**
- ⭐ Preference-based recommendation system
- 📊 Rating-driven ranking algorithm
- 📹 Real-time video calls (WebRTC + WebSockets)
- 💬 Messaging system (voice, images)
- 🔄 Request / accept / refuse workflow

**Architecture**
- **Backend:** NestJS microservices
- **Frontend:** Angular dashboard
- **Mobile:** React Native / KMP clients
- **Database:** PostgreSQL + MongoDB
